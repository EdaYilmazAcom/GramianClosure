# GramianClosure

Implementation and experimental study of nonlinear closure techniques for moment systems based on orthogonal polynomials.

## Publications

This repository contains code accompanying the paper:

- **Nonlinear Closures for Moment Equations Based on Orthogonal Polynomials**  
  arXiv preprint, 2024.  
  https://doi.org/10.48550/arXiv.2407.05894
